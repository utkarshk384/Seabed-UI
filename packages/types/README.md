## @seabedui/types
---

These are types that anyone who is using typescript can use to import types in their project.
### 🐟 &nbsp; Install

```sh
yarn add @seabed/types
```
or
```sh
npm install --save @seabed/types
```


### 🐬 &nbsp; Usage

```typescript
import { ThemeType } from '@seabedui/types';

const theme:ThemeType = {}
```
